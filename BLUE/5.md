## 数组
### map  --- 映射 --- 一个对一个
- 每个参数都要执行一次`map`里面的函数
- map()方法的三个参数
>第一个参数. item（成员）
>第二个参数. index（索引）
>第三个参数. arr（原数组）
```
let arr = [20,30,50,100];
arr.map(function(item,index,arr){
    return item>=50?"大于等于50":"小于等于50";
})
```

### reduce --- 汇总 --- 一堆出来一个
- 也是每个参数都要执行一次reduce()里面的函数
- reduce()方法三个参数
>第一个参数:tmp (总和)
>第二个参数:item (成员)
>第三个参数:index (索引)
```
//总数
[300,50000,6888] => 57051
//算平均数
[12,59,99] ==> 56.67
```
